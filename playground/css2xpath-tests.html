About tests

All tests are run in your current browser.

Tests Passed mean that the elements selected by CSS selector are reference equal (not just only by count) to the elements selected by the generated XPath.

Note that XPath implementation for the class attribute is the standard one (like other attributes). See Non-standard XPath behavior of class attribute

Not valid selectors are mostly from an extended set of selectors that the browser doesn't support, but are converted to valid XPath.
